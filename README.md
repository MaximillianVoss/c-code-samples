# Original C Array Exercises (2016)

This unrelated archive branch reconstructs two source-only course stages in
their filesystem chronology:

1. `Lab 5` (2016-04-12);
2. `lab1изменение размера массива` (2016-04-27).

Tag `original-c-array-exercises-2016` points to the final byte-exact source
snapshot before preservation documentation was added. Original names, sizes,
SHA-256 values, and NTFS timestamps are recorded in
`preservation/file-metadata.csv`.

Generated IDE databases and build outputs are intentionally absent. Their
metadata and selected deliverable payloads remain in the private
`legacy-programming-projects-archive` repository. The active, tested C17
refactor lives on `c-code-samples/main` under
`Legacy/c-array-exercises-2016`.
