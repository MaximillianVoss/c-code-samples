# C Array Exercises (2016)

Две лабораторные работы на C восстановлены из старой папки `Архив`:

- `Lab 5` - сортировка каждой строки динамической матрицы;
- `lab1изменение размера массива` - увеличение и уменьшение динамического массива.

Каталог `legacy` хранит исходные Visual Studio 2015 файлы побайтно, включая
первоначальные русские имена. Полный реестр исходных файлов, NTFS-дат,
размеров и SHA-256 находится в `preservation/file-metadata.csv`. Старые
`.sdf`, `.VC.db`, объектные файлы и сборки не дублируются здесь: их метаданные
и, где это оправдано, payload сохранены в private-репозитории
`legacy-programming-projects-archive`.

## Рефакторинг

Переносимое ядро в `src` исправляет основные проблемы исходников:

- изменение размера не теряет старый указатель при ошибке выделения памяти;
- рост проверяется на переполнение и заполняет новые элементы нулями;
- уменьшение до нуля освобождает память и обнуляет указатель;
- строки матрицы сортируются с корректными размерами и без утечки строк;
- ошибки передаются явным кодом, а не остаются скрытыми в интерактивном UI.
